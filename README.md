## Description technique du projet

Le projet a été réalisé sous le framework PHP [Laravel](https://laravel.com/) et le framework [Livewire](https://laravel-livewire.com/) qui simplifie la création d'interfaces dynamiques dans Laravel.
Le design pattern de l'application est en Modèle Vue Contrôleur (MVC) et l'architecture est en REST.
